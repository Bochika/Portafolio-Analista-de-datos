# Proyectos de Análisis de Datos en Python

En esta carpeta encontrarás proyectos desarrollados en Python, enfocados en análisis, limpieza, visualización y modelado de datos. Cada subcarpeta corresponde a un proyecto específico, acompañado de su propio README explicativo.

## Tecnologías y librerías principales:

- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

## Ejemplos de posibles proyectos:

- Análisis exploratorio de datasets de ventas
- Modelos predictivos de demanda
- Visualización de tendencias y métricas

---

Para cada proyecto, encontrarás un README que detalla objetivos, herramientas utilizadas y principales resultados.
